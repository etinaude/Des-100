Components

This is a project to explore the basic functionality of Arduinos and LEDs. I started by collecting the components I needed for the project. These were:

  • 1 Arduino Uno Replica
  • 3 LEDs
  • 6 jumper wires
  • 1 low voltage button
  • 3 180Ω resistors
  • 3 micro USB cable

Once collecting these, I followed the circuit provided in the assignment details and ensured that the LEDs had the correct polarity. I decided to use 180Ω resistors to give minimally brighter glows while still preventing the burnout of the LEDs.


Innovations

LED example

I then copied the code from Appendix 3 and stepped through each script to ensure the hardware worked properly. I decided to use this opportunity for an Arduino and LED-based project I had already been planning. The plan was to create a lightbox which I could place on the wall above my computer and use as decor, and when I'm in a video conference, I could use it to light myself.

I started by testing an addressable LED strip to ensure it worked, I did this by using the fast LED library to code quickly and efficiently. I started by using the sample code provided by howtomechatronics and modified it to work with 25 LEDs (5*5 grid).


cut wires pattern

I then cut the led strip into strips of 5 LEDs, measure the distance between the LEDs and cut some braided core wire accordingly, which I soldered together in a "snake pattern" to minimise the length of wire I'd need. I then Laser cut:

  • a box designed in MakerCase from 3mm acrylic
  • a base plate for the LEDs unattached to the box
  • a diffusion sheet cut from a sheet of tracing paper since it has a nice texture and a good level of transparency

Code


Patterns

I created a range of functions to display the patterns for example: 


  void blueSpec(){
    for(int i = 0; i < 20; i++){              // Create 20 patterns
      for(int i = 0; i < NUM_LEDS; i++){      // Loop though LEDs and pick a random blue for each LED
        int s = random(120,230);              // Change Saturation
        int v = random(120,230);
        int h = 120 + random(0,30);           // Pick a hue close to blue
        leds[i] = CHSV(h, s, v);              // Set LED colour
      }
      delay(200);                             // Wait 0.2 seconds before creating a new pattern
      FastLED.show();                         // Display the colours
    }
  }


I then created a function to select which pattern to display using a switch statement. This way, I can pass a number as an index of which function to run. I then looped through all the functions continuously.

Future


With this setup, I can modify the box and code slightly to do a wide range of things.

  • play the functions in order or at random
  • I can include a switch to change between run the functions and white light for video conferencing
  • I can include an esp32 to allow wireless control and voice commands

I plan to expand and improve on this project lots. My first step would be to create a PCB with an esp32 and switch. From there, I'd add extra functions and a better wall mount mechanism.
